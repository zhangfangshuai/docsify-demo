### 三级目录
Javascript代码示例
```Javascript
// 防抖函数 (本案例利用闭包实现，这样无需再antiShake外部定义timeout变量)
function antiShake(fn, wait) {
    let timeout = null
    return args => {
        if (timeout) clearTimeout(timeout)
        timeout = setTimeout(fn, wait)
    }
}
```
